# Victor Remodeling 🏠

Sitio web profesional bilingüe (EN/ES) para **Victor Remodeling** — servicios de remodelación y reparación.

## ✨ Características

- ⚡ React 19 + Vite 8 — build ultra rápido
- 🌐 Bilingüe inglés / español con selector en el header
- 📱 Diseño 100% responsivo (mobile-first)
- 🎨 Paleta profesional azul marino / naranja
- 📧 Formulario de contacto con envío por email
- 🔗 Redes sociales (Facebook, Instagram)

## 🚀 Cómo usar

### Instalación
```bash
npm install
```

### Desarrollo local
```bash
npm run dev
```
Abre http://localhost:5173 en tu navegador.

### Build de producción
```bash
npm run build
```

### Preview del build
```bash
npm run preview
```

## 📁 Estructura
```
src/
├── components/    — Header, Hero, About, Services, Contact, Footer
├── context/       — LanguageContext (EN/ES)
├── App.jsx
├── main.jsx
└── index.css
```

## 🛠 Tecnologías
- React 19
